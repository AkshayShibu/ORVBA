ORVBA – On Road Vehicle Breakdown Assistance System

ORVBA is a web-based platform designed to assist vehicle users during emergency breakdown situations.  
The system connects users with nearby mechanics and fuel providers, enabling quick assistance when a vehicle breaks down on the road.

The project aims to provide a simple and efficient solution for roadside emergencies by allowing users to request mechanical support or fuel delivery through an easy-to-use platform.

Features

- User registration and login
- Mechanic support request
- Fuel delivery service
- Product browsing and purchase through shop module
- Admin management system
- Mechanic dashboard for managing service requests
- Fuel provider dashboard for delivery management
- Order and service tracking

System Modules

Admin
- Manage users, mechanics, and fuel providers
- Monitor service requests
- Manage shop products
- Platform administration

User
- Register and login
- Request mechanic support
- Request fuel delivery
- Browse and purchase products from shop

Mechanic
- Receive service requests
- View user breakdown details
- Accept and complete repair tasks

Fuel Provider
- Receive fuel delivery requests
- Deliver fuel to user location
- Update delivery status

Technologies Used

- Python
- Django Framework
- HTML
- CSS
- Bootstrap
- SQLite Database
